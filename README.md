# blackjack
Реализация карточной игры "Blackjack" на Pawn для SAMP/OPEN.MP сервера.
Реализация в виде include файла, требует y_hooks, также требует реализации функции AddPlayerMoney(playerid, amount) -> bool:status для взаимодействия с балансом игрока.
Все еще не финальный вариант, однако основные игровые механики реализованы
